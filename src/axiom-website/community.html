Discussion of math theory and philosophy related to Axiom

Axiom Releases

Axiom is released about every two months on a best-effort basis. The "golden" released sources are maintained in GIT, SVN, and CVS.

Axiom release (Git) is hosted on GitHub.

Axiom release (Git) is hosted on Savannah.

Axiom release (git) is hosted on Sourceforge.

Axiom Development

Axiom is continuously developed and a "silver" version is available. The development sources are maintained in git and SVN.

Axiom development (git) is hosted on Sourceforge.

Axiom development (git) is hosted on axiom-developer.org
